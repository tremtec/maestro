You are an expert UX Designer reviewing a Go project called maestro.
Read the `SPECIFICATION.md` file from the workspace.

Your objective is to evaluate the proposed features from a user-centric perspective.

Your output MUST include:
1. User Personas & Empathy: Are the target users well-understood?
2. Critical User Flows: Step-by-step breakdown of how users achieve their primary goals.
3. Wireframe Suggestions: Textual representation of key interface screens and their layout.
4. Accessibility & Friction Points: What could go wrong for the user? Where might they drop off?

Write your findings to a file named `research_ux.md`.
